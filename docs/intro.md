# Research Tech

Web技術の調査・検証結果をまとめるドキュメントサイトです。

## このサイトについて

各種Web技術について調査した内容を、再利用可能なドキュメントとして整理しています。

:::caution AIによる調査ドキュメントについて
本サイトのドキュメントは **Claude Code（Anthropic の AI）が100%調査・執筆** しています。人間による網羅的な事実確認は行っておりません。以下の点にご注意ください。

- **情報の正確性**: AIが Web 検索や公式ドキュメントを参照して作成していますが、すべての情報が正確であることを保証するものではありません
- **情報の鮮度**: 各ドキュメントの `last_update` に記載された日付時点の情報です。技術の進歩により内容が古くなっている可能性があります
- **出典の確認**: 重要な意思決定に利用する場合は、各ドキュメント末尾の参考リンクから一次情報を必ずご確認ください
:::

## カテゴリ構成

ドキュメントは技術カテゴリごとにディレクトリで分類されます。新しいカテゴリは調査テーマに応じて追加されます。

例: `react/`, `css/`, `performance/` など
